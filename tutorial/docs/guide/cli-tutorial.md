# 终端教程

WebUI 更适合第一次完整体验，而终端模式更适合开发、调试和脚本化使用。

这一页后续主要承载两类内容：

1. 整个 LoopAI 的终端启动教程
2. 各个子 Agent 的单独启动教程

## 整体启动

如果你想直接启动主流程，可以运行：

```bash
python examples/scripts/run_starter.py
```

这条命令更适合：

- 验证主流程配置
- 调试 Starter 调度逻辑
- 在没有 WebUI 的场景下直接体验任务流
